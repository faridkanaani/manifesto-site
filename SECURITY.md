# Security Policy

## Supported Versions

This project is a personal website. Security fixes are applied to the latest version on the `main` branch.

## Reporting a Vulnerability

If you discover a security issue, please report it privately:

- Email: farid.kanaani.official@gmail.com

Please do not open a public GitHub issue for security vulnerabilities.

## Scope

- Dependencies (npm packages)
- Build/deploy configuration
- Any XSS or injection risks

Thanks for helping keep this project safe.
