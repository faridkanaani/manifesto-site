# Zero Point Manifesto

The weirdest personal website in the world. A digital manifesto for a Code Artist.

## Live Site
https://faridkanaani.vercel.app/

## Features
- **Glitch Effects:** Hover on elements for RGB glitches and animations.
- **Terminal Interface:** Press 'T' to open a hidden terminal with manifesto content.
- **Breach Modal:** Click [ BREACH_DATA ] or [ CONNECT_VOID ] for project/contact info.
- **Easter Eggs:** Click the Eye icon for hidden messages.
- **Dynamic Content:** Random terminal messages and floating fragments.
- **Mouse Interactions:** Mouse-following effects and radial gradients.

## Tech Stack
- Next.js 14 (App Router)
- Framer Motion
- Tailwind CSS
- TypeScript
- Vercel Deployment

## Run Locally
1. Clone the repo:
   ```bash
   git clone https://github.com/faridkanaani/manifesto-site.git
   cd manifesto-site
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run dev server:
   ```bash
   npm run dev
   ```

4. Open http://localhost:3000

## Deploy on Vercel
1. Fork or clone this repo.
2. Connect to Vercel.
3. Deploy with one click.
