# Contributing to Zero Point Manifesto

Thanks for your interest in contributing to the Zero Point Manifesto! This project is open source and welcomes contributions from the community.

## How to Contribute

1. **Fork the repository** and create a feature branch.
2. **Make your changes** following the code style.
3. **Test locally** with `npm run dev` and `npm run build`.
4. **Commit with clear messages** and submit a pull request.

## Code Style
- Use TypeScript for type safety.
- Follow Next.js best practices.
- Keep components modular and reusable.
- Add comments for complex logic.

## Reporting Issues
- Use GitHub Issues for bugs or feature requests.
- Provide detailed steps to reproduce.
- Include browser/OS information.
